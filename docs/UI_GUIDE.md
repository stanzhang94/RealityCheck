# UI Guide

## Opening The Financial Manual

- [F] Default hotkey: `O`.
- [F] The key is configured by `OpenReportKey` in `config.json`.
- [F] `ModEntry.OnButtonsChanged` opens `FinanceMenu` only when the world is ready and no other clickable menu is active.

## Financial Manual Pages

Current source shows these report tabs or views:

- [F] Daily report.
- [F] Seasonal report.
- [F] Annual report.
- [F] Tax report.
- [F] Market Price report.
- [F] Market Price history chart for a selected item.
- [F] Income and expense details inside report pages.

## Market Price UI

- [F] Supports item, market price, base price, daily multiplier, and total multiplier columns.
- [F] Supports sorting.
- [F] Supports search.
- [F] Supports favorite markers backed by save data.
- [F] Supports history view for selected market entries.

## Exchange UI

- [F] Opened from the Financial Manual when exchange services are available.
- [F] Contains account, create-contract, and positions flows in current source.
- [F] Displays account history, balances, positions, chart/risk details, delivery actions, and confirmation modals.
